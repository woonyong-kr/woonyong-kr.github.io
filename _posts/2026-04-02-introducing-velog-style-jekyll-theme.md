이 저장소는 velog의 읽기 좋은 흐름을 참고하되, 정적 블로그 운영에 필요한 요소를 Jekyll 방식으로 다시 구성한 테마입니다.

기본으로 포함된 기능은 다음과 같습니다.

- velog 스타일의 글 목록과 시리즈 탭
- 태그 필터와 검색
- GitHub 프로필 동기화
- 기여 그래프 히어로 섹션
- GitHub Pages와 Jenkins 배포 흐름

이 데모에 포함된 프로필 아이콘, 사이트 아이콘, 글 썸네일은 모두 이 저장소를 위해 직접 만든 SVG 자산이라 외부 저작권 이슈 없이 샘플로 배포할 수 있습니다.

오픈소스 테마로 배포할 때는 이 글처럼 “무엇을 제공하는지”, “어떻게 설치하는지”, “어디를 수정하면 되는지”가 바로 보이도록 몇 개의 안내 글을 함께 넣어두는 편이 좋습니다.
