이 페이지는 소개 탭을 켰을 때 어떤 식으로 테마를 설명하면 좋은지 보여주는 예시입니다.

## 이 테마가 제공하는 것

- velog 스타일의 글 목록과 시리즈 탭
- 태그 필터와 검색
- GitHub 프로필 동기화와 기여 그래프
- GitHub Pages와 Jenkins 배포 흐름

## 어떤 블로그에 어울리는가

프론트엔드, 백엔드, 인프라, 개발 문화처럼 긴 글과 짧은 메모가 함께 올라오는 개인 개발 블로그에 잘 맞습니다.

## 커스터마이징 포인트

`_data/profile.yml`, `_data/theme.yml`, `_data/series.yml`만 바꿔도 브랜드, 탭, 소개 문구, 시리즈 구조를 대부분 조정할 수 있습니다.

## 오픈소스 배포 팁

이 페이지는 그대로 써도 되지만, 공개 테마라면 설치 방법과 설정 파일 위치를 짧게 정리하는 문서형 소개 페이지로 바꾸는 편이 더 좋습니다.
