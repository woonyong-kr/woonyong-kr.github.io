## API 응답 예시

```http
GET /api/posts?tag=frontend HTTP/1.1
Host: example.dev
Accept: application/json
```

```json
{
  "items": 12,
  "nextCursor": "post_014",
  "status": "ok"
}
```

```ts
export type PostSummary = {
  id: string;
  title: string;
  tags: string[];
  thumbnail?: string;
};
```

정적 사이트라도 검색과 필터는 충분히 빠르게 만들 수 있지만, 데이터가 커질수록 **초기 렌더 비용**과 **브라우저 메모리**를 같이 봐야 합니다.
